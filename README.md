# Koulu-projekti – T23 Versionhallinta

Tämä on **T23 Versionhallinta** -tehtävän harjoitusrepo.

## Sisältö
- Harjoitustiedostoja (esim. `ostoslista.txt`, `tarina.txt`)
- Commit-historiaa useilla muutoksilla

- `.gitignore`
- Branchien käyttö ja merge
